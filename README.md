# 💼 Portfolio

This is a personal portfolio built with [Next.js](https://nextjs.org/) and [Tailwind CSS](https://tailwindcss.com/). It showcases professional experiences, programming language skills, certifications, personal projects, and more.

🔗 **Live Preview**: [https://lucasheartcliff.com.br](https://lucasheartcliff.com.br)

## ✨ Features

* 🔍 **About** section with a personal introduction
* 📊 Interactive charts showing experience with programming languages (via [ApexCharts](https://apexcharts.com/))
* 👨‍💻 Career **timeline** with role progression
* 🎓 List of **certifications** from online platforms
* 📁 **Projects** section with GitHub links
* 🌐 Multilingual support (i18n)
* 📱 Fully responsive layout

## 🚀 Technologies Used

* [Next.js](https://nextjs.org/)
* [React](https://reactjs.org/)
* [Tailwind CSS](https://tailwindcss.com/)
* [TypeScript](https://www.typescriptlang.org/)
* [ApexCharts](https://apexcharts.com/)
* [i18next](https://www.i18next.com/) for internationalization

## 📦 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/lucasheartcliff/portfolio.git
```

2. Install the dependencies:

```bash
npm install
# or
yarn install
```

3. Start the development server:

```bash
npm run dev
# or
yarn dev
```

4. Open your browser and go to: `http://localhost:3000`

## 📚 Sections Overview

* **Languages**: visual representation of time spent on each programming language
* **Experiences**: timeline showing roles at companies like Intelie by Viasat and freelance work
* **Certifications**: courses completed on Udemy and Alura
* **Projects**: sample apps like `Crypt Image`, `Data Persistence`, and this portfolio itself
